smpl

"smpl" is an automated starter framework for sass lovers who like to design in the browser. I created this repo to help me get started with new projects. It includes some very basic UI elements and some helper classes I tend to use in every project.

You can check it out for the time being from https://github.com/simplethemes/smpl.git. I will be moving it to github shortly.

To use, you'll need to have node.js, grunt and SASS with Compass. Once you download, cd into the directory and run npm install. Run grunt-dev.command and you should be good to go. This workflow works best with Google Chrome with the LiveReload extension enabled.
